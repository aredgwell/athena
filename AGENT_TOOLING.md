# Agent Tooling Reference

Preferred local binaries for agent workflows.

## Bootstrap

Run:

```bash
task ai:tools:check
```

## Preferred Tools

- Search: `rg`
- JSON: `jq`
- YAML: `yq`
- HTML: `htmlq`
- Diffs: `difft`
- Task runner: `task`

## Optional Accelerators

- Context packaging: `repomix`
- Interactive selection: `fzf`
- Shell quality: `shfmt`, `shellcheck`

## Rule

Prefer structured output tools over ad-hoc parsing.
